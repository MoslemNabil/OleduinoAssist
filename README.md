# OLED Face Designer & Code Generator

A professional web-based tool for designing monochrome UI, animations, and faces for SSD1306 OLED displays. Convert drawings, images, and videos directly into C++ (Arduino/U8g2) or MicroPython byte arrays.

---

## 🚀 Features

- **128x64 Pixel Canvas**: Pixel-perfect drawing tools (Pen, Eraser, Clear).
- **Multi-Frame Animation**: Create sequences, adjust FPS, and preview in real time.
- **Media Engine**: Upload images or MP4 videos and auto-convert to 1-bit monochrome XBMP format.
- **Hardware Presets**: Built-in support for Arduino, ESP32, ESP32-C3, ESP8266, Raspberry Pi Pico, and CH552.
- **Dual Language Output**: Generates production-ready C++ (U8g2) and MicroPython code.
- **Cross-Platform Deployment**: Fully responsive and optimized for touch and mouse.

---
